Heroku

Requirements

$ wget -qO- https://cli-assets.heroku.com/install-ubuntu.sh | sh

Create Heroku App

$ heroku login
$ heroku create example-project-name
$ heroku config:set DISABLE_COLLECTSTATIC=1

Heroku Web: Settings Tab -> Reveal Config Vars -> Add: SECRET_KEY : secret key value

Heroku Web: Deploy Tab -> Connect to Github -> Connect -> Enable Automatic Deploys

$ git push heroku master
